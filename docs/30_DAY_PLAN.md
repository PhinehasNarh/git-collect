# 30-Day Execution Plan

Assumes ~20 to 40 minutes per day. "Badge progress" notes which achievement the
day advances. Verify each on your profile (**github.com/PhinehasNarh** →
Achievements) the day after the triggering action, since GitHub awards on a
delay.

| Day | Action | Expected badge progress | Verification |

## What 30 days realistically gets you

- **Earned with certainty:** Quickdraw, YOLO, Public Sponsor, Pull Shark tier 1
  and tier 2, Pair Extraordinaire tier 1 (likely tier 2 if you pair often).
- **Earned with a willing collaborator and a public repo:** Galaxy Brain tier 1,
  likely tier 2.
- **Started but not guaranteed in 30 days:** Starstruck. Real stars come on
  their own schedule; do not force it.
- **Not earnable at all:** Arctic Code Vault, Mars 2020 Helicopter (retired).
